Overview

This is a fitting log created with the purpose of informing 4 “new” fits for the BaHPO4 data. These fits alter parameters of mask type and number of lorentzians. There are 2 different types of masks used: the first masks a single Bragg peak – only the largest, the second masks all Bragg peaks. Both are different from previous masking methods, in which the automatically generated Bragg peak mask in Mslice was used. Resolution data was created as well, with resolution functions for each temperature and masking method, totaling 14 resolution functions. These are all stored in the “resolution” folder. All fitting in the folders is saved in the fits folder, and according subfolders according to the mask used and the number of lorentzians fit. All data is corrected with vanadium, and fit with a delta curve, background curve, and Lorentzian(s) (one or two). In the this document, the number of lorentzians and type of mask used is specified directly after the fitting procedure heading. The table of contents is included with the intent of easy navigation.
